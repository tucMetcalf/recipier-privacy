Excel / CSV Import

Import a full recipe library from a spreadsheet.

Column Layout

Each row is one recipe. Columns must be in this order:

ColumnHeaderDescription
ACategoryRecipe category (e.g. Breakfast, Dinner). Can be blank.
BTitleRecipe name. Required.
CIngredientsIngredient list — one ingredient per line within the cell. Include amount and unit if known.
DDirectionsStep-by-step instructions — one step per line within the cell.
ECollectionOptional. Assigns the recipe to a named collection (created if it doesn't exist). Overrides the sheet-name default.

Example Row

Category  | Title            | Ingredients                          | Directions                      | Collection
Dessert   | Lemon Drizzle    | 200g butter                          | 1. Preheat oven to 180°C        | Baking
          |                  | 200g caster sugar                    | 2. Cream butter and sugar       |
          |                  | 3 eggs                               | 3. Add eggs one at a time       |
          |                  | 200g self-raising flour              | 4. Fold in flour                |
          |                  | Zest of 2 lemons                     | 5. Pour into tin and bake 45min |

Sheet Names & Collections

Each sheet tab in an Excel file is treated as a separate collection by default. The sheet name becomes the collection name.

  • Sheet "Breakfast" → all recipes go into a "Breakfast" collection
  • Sheet "Dinner" → all recipes go into a "Dinner" collection
  • Use column E (Collection) to override this per row
💡 Tip: CSV files have only one sheet. Use column E to assign recipes to different collections within a single CSV.

Side-by-Side Ingredient Sections

Some recipes (e.g. pastry + filling) have two ingredient lists that sit side-by-side in the spreadsheet. Recipi'er detects this automatically.

Put both section headers on the same row — one in the left half of column C, one in the right half — and list ingredients below each:

PASTRY                      FILLING
150g margarine              675g pie apples, grated
250g plain flour             25g sugar
25g ground almonds           2 tbsp raisins
75g caster sugar             2 tbsp cream

Both sections will be imported as separate ingredient groups.

Supported File Types

XLSX Excel (recommended)

XLS Older Excel format

CSV Comma-separated values

GSHEET Google Sheets (share as "Anyone with the link can view" and paste the sheet URL)

Tips

  • Each row with a Title is imported as a separate recipe — even if Category and other fields are blank.
  • Blank rows between recipes are ignored.
  • For multi-line cells in Excel, use Alt+Enter (Windows) or Option+Return (Mac) to add a line break within a cell.
  • Duplicate recipes (matching an existing title in the same collection) are skipped.
