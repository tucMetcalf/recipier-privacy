Bulk URL Import

Import multiple recipe links at once from a plain text file.

Simple Format — One URL per Line

The simplest format: paste one recipe URL per line into a .txt or .csv file.

https://www.allrecipes.com/recipe/spaghetti-bolognese
https://www.bbcgoodfood.com/recipes/chocolate-cake
https://www.taste.com.au/recipes/beef-stroganoff

Advanced Format — With Category & Collection

Add a header row to assign a Category and/or Collection to each recipe. Columns can be in any order — they are matched by name.

ColumnDescriptionExample
URLRequired. The recipe link.https://example.com/recipe
CategoryOptional. Recipe category.Dinner
CollectionOptional. Adds the recipe to a named collection (created if it doesn't exist).Family Favourites
URL,Category,Collection
https://www.allrecipes.com/recipe/spaghetti,Dinner,Italian Nights
https://www.bbcgoodfood.com/recipes/chocolate-cake,Dessert,Baking
https://www.taste.com.au/recipes/beef-stroganoff,Dinner,
https://www.example.com/soup,,Quick Meals
💡 Tip: Leave Category or Collection blank for a row and those fields will be skipped. Columns can be in any order as long as the header names match.

Supported File Types

TXT One URL per line (simple format only)

CSV Comma-separated values, with or without header row

Tips

  • Recipes are imported one at a time — the screen shows progress as each URL is processed.
  • Failed rows are shown with an error and can be retried individually.
  • Duplicate URLs that are already in your library will be skipped.
  • Social media URLs (Facebook, Instagram, TikTok) work just like entering them manually.
